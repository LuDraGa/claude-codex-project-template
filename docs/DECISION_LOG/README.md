# DECISION_LOG

Architecture Decision Records (ADRs) for this repository.

## Naming Convention

`YYYY-MM-DD-####-short-title.md`

- `YYYY-MM-DD`: decision date
- `####`: monotonic sequence (`0001`, `0002`, ...)
- `short-title`: concise kebab-case summary

## ADR Index

- `2026-03-03-0001-ledger-outbox-and-lease-architecture.md` (Accepted)

## Required Sections

- Context
- Options Considered
- Decision
- Consequences

Use `0000-template.md` to start a new ADR.
